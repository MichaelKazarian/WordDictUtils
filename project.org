* Технічне завдання
** Загальний опис проекту
. [Цей проект](https://github.com/MichaelKazarian/WordDictUtils) призначено для створення словників для проекта [WordDolphin](https://play.google.com/store/apps/details?id=com.waverunner.wordrunner).
Основа проекту мій інший проект, [WordDictCore](https://github.com/MichaelKazarian/WordDictCore) який презентує відкритий формат для зберігання слів для додатку WordDoplhin.
Словники будуть створюватись через утіліти командного рядка. ось pom.xml Для WordDictUtils. Залежить від

**  Призначення проекту
Проект WordDictUtils — це набір утиліти командного рядка, призначених для:

Генерації словників у форматі .wdi (відкритий формат WordDictCore)
Конвертації даних з зовнішніх джерел (Wiktionary, CSV, JSON тощо) у формат WordDolphin
Валідації та перевірки цілісності словників
Оптимізації та стиснення словників
Підтримки багатомовних словників (EN→UK, UK→EN, тощо)

Словники, створені за допомогою цих утиліт, використовуються у мобільному додатку WordDolphin.

	 @echo " make clean       — Clean build artifacts"

	 @echo " make deploy      — Deploy to GitHub Packages (if configured)"

	 @echo " make all         — Clean, build, test, and package"
	 @echo ""

 # ------------------------
 # Build, Test, Clean
 # ------------------------

 build:
	 mvn clean package -DskipTests=false

 test:
	 mvn test

 clean:
	 mvn clean

 # ------------------------
 # Install / Deploy
 # ------------------------

 install:
	 mvn install

 deploy:
	 mvn deploy

 # ------------------------
 # Run (with main class)
 # ------------------------

 run:
	 @if [ -z "$(MAIN)" ]; then \
	   echo "Please specify MAIN class, e.g.:"; \
	   echo " make run MAIN=com.worddict.wiktionarybot.WiktionaryEnglish"; \
	 else \
	   mvn exec:java -Dexec.mainClass=$(MAIN); \
	 fi

 # ------------------------
 # Utility targets
 # ------------------------

 jar:
	 @echo "Built JAR file: $(JAR_FILE)"

 all: clean build test jar
